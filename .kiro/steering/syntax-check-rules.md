# HTML 파일 문법 검사 규칙

## 필수 검사 항목
1. **HTML 구조 검사**
   - 모든 태그가 올바르게 닫혔는지 확인
   - 속성값이 올바르게 인용부호로 감싸졌는지 확인

2. **CSS 검사**
   - CSS 변수 선언이 올바른지 확인
   - 선택자 문법이 정확한지 확인

3. **JavaScript 검사**
   - 템플릿 리터럴 백틱 이스케이프 확인
   - 함수 선언과 호출이 일치하는지 확인

## 오류 발견 시 행동
- 즉시 수정하고 사용자에게 알림
- 수정 후 다시 문법 검사 수행
- 모든 오류 해결 후 Git 커밋
