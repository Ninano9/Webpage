br : 띄어쓰기 마지막에 '/' 필요없음
p : 단락 나눔
strong : 굵은 글씨
u : 밑줄
img src(image source) : img scr="이름.jpg"이미지 삽입, 100%(화면크기에 맞춤)
img scr="하위폴더/이름.jpg" : 이미지가 하위폴에 있을 경우
h1~h6 : 주로 제목, 큰글씨+한줄띄우기, 숫자가 작을수록 글씨 커짐
li : list 목록 만들기
ul : Unordered list li의 부모로 연관 항목 묶기
ol : Ordered list li 목록에서 순서 숫자 자동 생성
title : 제목 생성
meta charset="utf-8" : 브라우저가 파일을 읽을 때 ""안에 있는 형식으로 읽을 수 있게 하는 명령어
html : head와 body를 묶는 최상단 명령어
!doctype html : 관용적 표현, 파일이 html이라는 뜻
style background-image : 배경 이미지 삽입
background-repeat : no repeat; : 사진 반복 없음
!-- -- : 주석
xmp : 코드 내용 그대로 보여주기

a href : anchor hypertext reference, 링크를 걸고 싶을 때 사용
a 새링크 열기 : target="_blank"("_"중요)
a 미리보기 : title="설명"
